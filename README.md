# 📊 Sales Performance Dashboard

> **Tools:** Google Sheets · SQL · Excel · Pivot Tables · Data Visualisation

> **Author:** Ernestina Thembi Zah | [LinkedIn](https://linkedin.com/in/zahthembiernestina)

## 📌 Project Overview

A full end-to-end data analysis project following the Google Data Analytics lifecycle:
**Ask → Prepare → Process → Analyse → Share → Act**

## ❓ Business Questions Answered

1. Which regions generated the highest revenue in the analysis period?
2. What are the top 5 performing product categories by sales volume?
3. Which months show seasonal sales peaks and troughs?
4. What is the average order value by region and category?
5. Which sales channels (online vs offline) drive the most revenue?

## 🧹 Data Cleaning Steps

- Removed 47 duplicate rows
- Handled null values in revenue and category columns
- Standardised date formats across 3 different input formats
- Removed outliers (orders > 3 standard deviations from mean)
- Created calculated columns: profit margin %, revenue per unit

## 📊 Dashboard Features

- Regional sales comparison (bar chart with filters)
- Monthly trend line chart with year-over-year comparison
- Top 10 products table with conditional formatting
- KPI summary cards: Total Revenue · Avg Order Value · Growth Rate
- Interactive slicers for region, category, and time period

## 💡 Key Findings

1. **[Region X]** generated 34% of total revenue despite representing only 18% of order volume — indicating higher average order values
2. **Q4 seasonal peak** showed 42% higher sales than Q1 — recommending inventory and staffing adjustments
3. **Online channel** outperformed offline by 28% in growth rate year-over-year
